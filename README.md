# Java Calculator

A simple calculator developed in Java to practice programming logic fundamentals, including:

- Conditional statements (if/else)
- Loops
- User input handling
- Basic arithmetic operations

## Requirements

- JDK 21

## Running the Project

Compile the source file:

```bash
javac Calculate.java
```

Run the Application:

```bash
java Calculate
```

## Purpose

This project was created as a study exercise to improve problem-solving and programming logic skills using Java.
